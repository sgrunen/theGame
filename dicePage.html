Ability and Skill Rolls

The basis of the mechanins for the system are skill tests. A roll is used to determine the chance for success and the eventual outcome of many tasks a character attempts.

The rolls the character makes are modified by their skills and abilities and how they are test against any opposition. If Bob wants to batter through the castle gates, or push down the castle wall, he may be alloed a roll to try. Is he strong enough? Only tie, lots and lots of time, will tell.

The most basic roll in the Game is versus an attribute or skill. To do this, subtract the pertinent attribute or level from 20. That result is the base target level, and a successful roll is that number or higher.

Base Target Number = 20 - Attribute or Skill Level

The Base Target Number can be modified by a hoard of different things. The most common is a difficulty number which is set by the GM. Another common modifier is Defense Value. Yep, combat is based on skills too. A Base Target Number plus known modifiers is the Target Number. The number the player rolls on the dice (normally a 20 sided dice) minus the Target Number is the result.

Target Number = Base Target Number + Modifiers

Result = Target Number - d20 roll
